Perintah Dasar Git yang Wajib Dihafal

Ini adalah cheatsheet atau contekan untuk perintah-perintah Git yang paling sering digunakan.

Perintah Deskripsi
git clone [url] Menyalin repository dari remote (misal: GitHub) ke komputer lokal.
git status Melihat status file di dalam repository (apakah ada perubahan, file baru, dll).
git add [nama-file] Menambahkan satu file ke area 'staging' untuk disiapkan sebelum di-commit.
git add . Menambahkan SEMUA file yang berubah ke area 'staging'.
git commit -m "pesan commit" Menyimpan perubahan yang ada di 'staging' ke riwayat repository dengan sebuah pesan.
git push Mengirim semua commit dari lokal ke repository remote (GitHub).
git pull Mengambil semua perubahan terbaru dari repository remote ke lokal.
